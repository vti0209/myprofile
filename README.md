# Craftivo - Professional IT Portfolio

A premium, high-performance personal portfolio website designed for **Hồ Văn Tiết**, an Information Technology student specializing in Frontend Development and Business Analysis.

![Hero Section](assets/img/profile/Anhbia.jfif)

## 🌟 Key Features

- **Premium Design Aesthetics**: A sleek, modern "Cyber-Earth" theme (Mệnh Thổ) combining deep earthy tones with high-tech neon accents.
- **Dynamic Background Effects**:
  - Global floating technology icons (HTML, CSS, JS, PHP, etc.).
  - Interactive background grid with pulsing animations.
  - CRT-style scanline overlay for a professional digital feel.
- **Multilingual Support**: Fully integrated toggle between **Vietnamese** and **English** with localized content for all sections.
- **Advanced Portfolio Showcase**:
  - AI-generated project mockups.
  - Detailed tech stack badges.
  - Interactive cards with smooth hover transitions and GLightbox support.
- **Interactive Resume**:
  - Structured education and skills timeline.
  - Downloadable professional CV (PDF integration).
- **Responsive & Performance Optimized**: Built with Bootstrap 5 and AOS for smooth, mobile-friendly scrolling animations.

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **CSS Framework**: Bootstrap v5.3.8
- **Animations**: [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/), [Typed.js](https://mattboldt.github.io/typed.js/)
- **UI Components**: 
  - [GLightbox](https://biati-digital.github.io/glightbox/) (Images & Media)
  - [Swiper](https://swiperjs.com/) (Touch sliders)
  - [Bootstrap Icons](https://icons.getbootstrap.com/)
- **Custom Scripts**: 
  - `tech-effects.js`: Custom global floating icon system.
  - `main.js`: Core template logic and initialization.

## 🚀 Getting Started

### Prerequisites
A modern web browser (Chrome, Firefox, Safari, or Edge).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/vti0209/Craftivo.git
   ```
2. Open `index.html` in your browser.

## 🎨 Theme Customization (Mệnh Thổ)
The project utilizes CSS variables for easy theme management. To modify the core colors, update the `:root` section in `assets/css/main.css`:

```css
:root {
  --background-color: #0f0d0a; /* Dark Earth */
  --accent-color: #d4a373;     /* Warm Gold */
  --heading-color: #fdf0d5;    /* Creamy Sand */
}
```
